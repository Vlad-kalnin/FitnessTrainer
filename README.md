Telegram-бот, который помогает достичь спортивных целей. Использует gpt-oss-20b в Yandex Cloud для общения и дает персонализированные советы по фитнесу. Стадия - прототип на основе собственной идеи.

# Возможности

- Персональные консультации по фитнесу
- Сохранение истории диалога для каждого пользователя
- Проверка противопоказаний перед началом тренировок
- Дружелюбное общение в стиле личного тренера

# Требования

- Python 3.8+
- Telegram Bot Token
- Yandex API ключ и Project ID

# Установка

1. **Клонируйте репозиторий:**

```bash
git clone <url-вашего-репозитория>
cd <название-папки>
```

2. **Установите зависимости:**

```bash
pip install -r requirements.txt
```

3. **Создайте файл `requirements.txt` со следующим содержимым:**

```
python-telegram-bot
python-dotenv
openai
```

# Настройка

1. **Создайте бота в Telegram:**

   - Найдите [@BotFather](https://t.me/botfather) в Telegram
   - Отправьте команду `/newbot`
   - Следуйте инструкциям и получите токен бота

2. **Получите доступ к базовой модели Yandex Cloud:**

   - Зарегистрируйтесь в [Yandex Cloud](https://cloud.yandex.ru/)
   - Создайте API-ключ в разделе "Сервисные аккаунты"
   - Скопируйте Project ID из настроек проекта

3. **Создайте файл `.env` в корне проекта:**

```env
TELEGRAM_TOKEN=ваш_токен_telegram_бота
API_KEY_YANDEX=ваш_api_ключ_yandex
PROJECT_YANDEX=ваш_project_id
```

# Запуск

Запустите бота командой:

```bash
python main.py
```

После запуска вы увидите сообщение: `Бот запущен...`

# Использование

1. Найдите вашего бота в Telegram по имени, которое вы указали при создании
2. Нажмите `/start` для начала общения

# Примечания

- История диалогов сохраняется в папке `sessions/` для каждого пользователя отдельно
- Бот запоминает последние 20 сообщений для контекста
- Максимальная длина ответа — 500 токенов
